## Sample Run

| Language        | Steps                                                                  |
| --------------- | ---------------------------------------------------------------------- |
| Python          | <pre>python [filename.py]</pre>                                        |
| C               | <pre>gcc [filename.c]<br>./a.out  # unix<br>a.exe  # windows</pre>     |
| C++             | <pre>g++ [filename.cpp]<br>./a.out # unix<br>a.exe # windows</pre>     |
| Java            | <pre>javac [filename.java]<br>java [filename]</pre>                    |
